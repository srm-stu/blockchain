ABOUT THE SEMINAR

National Level Seminar on Blockchain Technology for IoT: An Information Security Perspective

The world of Internet of Things (IoT) is rapidly evolving, connecting billions of devices across various industries, from healthcare and transportation to agriculture and smart cities. While IoT promises unprecedented levels of connectivity and automation, it also brings significant security concerns, particularly in terms of device authentication, data integrity, and privacy. Traditional security mechanisms often fall short in IoT environments due to their centralized nature and vulnerability to attacks.

Blockchain technology, with its decentralized, immutable, and transparent nature, has emerged as a potential solution to address these challenges. However, IoT systems are not just isolated networks; they are often connected to cloud platforms and rely heavily on communication through complex network infrastructures. Thus, the integration of blockchain with IoT, cloud computing, and network infrastructures offers a comprehensive solution to enhance security, scalability, and trust within IoT ecosystems.

The Two Days National Level Seminar on Blockchain Technology for IoT: An Information Security Perspective will delve into how blockchain can be leveraged to secure IoT networks, cloud infrastructures, and interconnected systems. This seminar will provide a platform to explore how blockchain enhances security in IoT networks, how it can be applied in cloud environments for data integrity and privacy, and how it can solve scalability and interoperability challenges in large-scale IoT systems.

THEMES FOR STUDENTS PROJECT DEMO:

  • 1. Blockchain-Enhanced IoT Network Security: Distributed Threat Detection

    Project Overview
    - Develop a decentralized network security system using blockchain to securely log and verify potential threats and attacks within an IoT network.
    Key Features
    - IoT device event monitoring (e.g., intrusion, malfunction, anomalies)
    - Blockchain ledger for secure event logging and auditing
    - Decentralized threat detection and automated responses via smart contracts

  • 2. Blockchain-Based Secure IoT Network Communication Protocol

    Project Overview
    - Implement a secure communication protocol for IoT networks that uses blockchain to verify the authenticity and integrity of data packets transmitted between IoT devices..
    Key Features
    - Blockchain-based keys for communication authentication
    - Immutable logs for communication events
    - End-to-end encrypted data transfer across IoT network nodes

  • 3. Decentralized Network Management for IoT using Blockchain

    Project Overview
    - Build a decentralized network management platform for IoT devices, where network configurations, access control policies, and performance metrics are managed on a blockchain.
    Key Features
    - Blockchain for managing device access and network configuration
    - Decentralized monitoring of IoT devices and network health
    - Smart contracts for automating network management tasks

  • 4. Blockchain-Based IoT Network Access Control System

    Project Overview
    - Create a network access control system using blockchain to manage and validate access requests to an IoT network, ensuring only authorized devices can connect.
    Key Features
    - Blockchain for device identity and authentication management
    - Real-time access control via smart contracts
    - Immutable logs of access requests and decisions

  • 5. IoT Network Privacy and Data Protection Using Blockchain

    Project Overview
    - Implement a privacy management system for IoT networks, where sensitive data is encrypted and stored on the blockchain, allowing users to control who accesses their data.
    Key Features
    - Blockchain-based data encryption and privacy management
    - User-controlled access to sensitive data
    - Transparent audit trails for data usage and sharing
